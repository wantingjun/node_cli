## 自定义终端命令的过程
1. 创建index,js文件，添加shebang
2. 修改packag.json文件，增加bin
```
  "bin": {
    "why": "index.js"
  },
```
* 在终端输入`npm link`,之后执行why他，就可以执行`index.js`文件
